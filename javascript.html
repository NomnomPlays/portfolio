Enabling Javascript

Safari (iOS)

To enable Javascript on Safari for iOS, you first need to go to settings, scroll down until you find "Safari" then tap it. Next you need to scroll down in the Safari section until you see "Advanced" and tap it. Set the toggle for "Javascript" to enabled.

Chrome (Mobile)

To enable Javascript on the Chrome mobile app, you want to tap the 3 dots at the bottom of the page (•••) then tap on "Settings", then "Content Settings", then "Javascript". (If you don't have the option for it, it should be enabled, if not enabled, you can look around on other places)

Chrome (Web)

To enable Javascript on Chrome, go up to the address bar and click the lock icon. Click Javascript and enable it.

uBlock Origin

To enable Javascript using uBlock Origin, find the extension icon in the extensions tab. Next, click the icon. If the "</>" icon has an X over it, then Javascript is disabled and click the "</>" icon to enable Javascript.
